Política de Privacidad

Última actualización: 4 de febrero de 2026

TheosVisor es una aplicación desarrollada por Galapagos Tech ("nosotros", "nuestro") para la consulta de precios y gestión de imágenes de productos en establecimientos comerciales.

1. Información que recopilamos

TheosVisor no recopila, almacena ni transmite datos personales de los usuarios. La aplicación funciona como un visor de productos que se conecta al servidor interno de su empresa.

Datos técnicos

  • Códigos de barras escaneados: Se envían al servidor configurado por el usuario para consultar información de productos. No se almacenan de forma permanente en el dispositivo.
  • Imágenes de productos: Las fotos tomadas o seleccionadas se procesan localmente en el dispositivo y pueden ser enviadas al servidor de la empresa del usuario. No se envían a terceros.
  • Configuración del servidor: La dirección del servidor, protocolo y clave API se almacenan localmente en el dispositivo del usuario.
  • Caché de imágenes: Las imágenes de productos se almacenan temporalmente en el dispositivo para mejorar el rendimiento.

2. Uso de la cámara

La aplicación solicita acceso a la cámara del dispositivo para:

  • Escanear códigos de barras de productos
  • Tomar fotografías de productos para actualizar su imagen en el sistema

Las imágenes capturadas se procesan localmente y solo se envían al servidor interno configurado por el usuario.

3. Uso de la galería de fotos

La aplicación puede solicitar acceso a la galería de fotos para seleccionar imágenes existentes de productos. Las imágenes seleccionadas se procesan localmente.

4. Almacenamiento de datos

Todos los datos se almacenan localmente en el dispositivo:

  • Preferencias de configuración (SharedPreferences)
  • Caché de imágenes de productos (almacenamiento temporal)

No utilizamos servicios de almacenamiento en la nube ni bases de datos remotas propias.

5. Comunicación con servidores

La aplicación se comunica únicamente con el servidor configurado por el usuario (típicamente un servidor interno de la empresa). No se realizan conexiones a servidores de terceros, servicios de analítica, redes publicitarias ni servicios de rastreo.

6. Terceros

TheosVisor no comparte datos con terceros. No utilizamos:

  • Servicios de analítica (Google Analytics, Firebase, etc.)
  • Redes publicitarias
  • Servicios de rastreo o tracking
  • Redes sociales

7. Seguridad

La comunicación con el servidor puede realizarse mediante HTTPS según la configuración del usuario. Recomendamos configurar siempre el protocolo HTTPS para proteger la transmisión de datos.

8. Derechos del usuario

El usuario puede en cualquier momento:

  • Revocar los permisos de cámara y galería desde la configuración del dispositivo
  • Eliminar la caché de imágenes desde la configuración de la aplicación
  • Desinstalar la aplicación para eliminar todos los datos almacenados localmente

9. Niños

TheosVisor no está dirigida a menores de 13 años. Es una herramienta de uso comercial/empresarial.

10. Cambios en esta política

Nos reservamos el derecho de actualizar esta política de privacidad. Cualquier cambio será publicado en esta página con la fecha de actualización correspondiente.
